Mission: Video Reflection Prompt

Create a safe prompt for reviewing practice videos without pretending AI is a certified coach.

Time

55 min

One focused build sprint.

AI Skill

Observation prompts

Practice it immediately.

Output

Video review checklist

Something useful by the end.

Proof

Demo note

Show what changed or improved.

Mission timeline

0–10

Define limits

List what AI can observe and what it should not claim.

10–25

Create checklist

Build a checklist for posture, consistency, effort, and visible patterns.

25–40

Test with description

Use a written description of a practice video to test the prompt.

40–50

Add safety language

Make the prompt avoid medical or expert claims.

XP log

Save final video review prompt.

Starter prompt

Help me review a sports practice video description. Do not make medical claims or pretend to be a certified coach. Identify visible patterns, effort, consistency, and 2 questions I should ask a real coach. Here is the description: [paste description].
