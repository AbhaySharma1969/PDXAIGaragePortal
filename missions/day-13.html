Mission: Algebra Helper That Teaches

Build a math helper that asks questions, gives hints, and explains mistakes instead of just giving final answers.

Time

55 min

Short, focused, and useful.

AI Skill

Step-by-step tutoring

One school-safe skill.

Output

Algebra tutor prompt

A practical study artifact.

Rule

Teach, don't cheat

AI supports learning, not dishonesty.

Mission timeline

0–10

Define cheating boundary

Agree what help is allowed: hints, explanations, similar examples, mistake checks.

10–25

Build tutor prompt

Create a no-final-answer-first algebra prompt.

25–40

Test 3 problems

Use the tutor on equations, graphing, or word problems.

40–50

Improve the prompt

Add rules for asking Sahil to try before showing steps.

50–55

XP log

Save the tutor prompt and one corrected mistake.

Starter prompt

Act as my algebra tutor. Do not give the final answer first. Ask me what step I would try. If I am stuck, give a hint. If I make a mistake, explain it simply and let me try again. Use a similar example before solving my exact problem.

Done means done

  • Algebra tutor prompt exists.
  • One test example is completed.
  • One screenshot or note is saved.
  • One thing to improve is recorded.
  • 60 XP is logged.
