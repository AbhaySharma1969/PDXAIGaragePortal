Mission: Study Guide Generator

Turn class notes into a useful study guide with key terms, practice questions, weak areas, and a short review plan.

Time

55 min

Short, focused, and useful.

AI Skill

Summarization + quiz design

One school-safe skill.

Output

Study guide template

A practical study artifact.

Rule

Teach, don't cheat

AI supports learning, not dishonesty.

Mission timeline

0–10

Pick a school topic

Choose Spanish, algebra, science, English, or history notes.

10–25

Generate study guide

Ask AI for key ideas, definitions, examples, and common mistakes.

25–40

Create practice questions

Generate multiple choice, short answer, and explain-your-thinking questions.

40–50

Rank weak areas

Ask AI to identify what Sahil should review first.

50–55

XP log

Save the study guide template and one practice quiz.

Starter prompt

Turn these notes into a study guide. Include key terms, simple explanations, examples, common mistakes, 10 practice questions, answer key, and a 20-minute review plan. Do not add facts unless clearly labeled as extra context.

Done means done

  • Study guide template exists.
  • One test example is completed.
  • One screenshot or note is saved.
  • One thing to improve is recorded.
  • 55 XP is logged.
