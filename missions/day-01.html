Day 1 · 40 XP

Mission: Build the Garage Identity.

Today’s goal is to define what PDX AI Garage is, create the first founder profile, and make one reusable prompt card.

Time

55 min

Short enough to stay fun.

Build

Founder profile

A clear identity for Sahil as the first teen founder.

AI Skill

Prompt basics

Role, task, context, format.

Proof

Screenshot + note

Capture what worked.

Mission timeline

0–10

Warm start

Ask: “What would make AI learning actually interesting for Sahil?” Write 5 answers without judging them.

10–25

AI brainstorm

Use AI to generate 10 possible mission themes for PDX AI Garage. Keep the ones connected to LEGO, sports, e-bikes, school, and real projects.

25–40

Build founder profile

Create Sahil’s first founder profile: interests, AI goals, project ideas, and what he does not want this to become.

40–50

Create first prompt card

Write one reusable prompt that helps Sahil learn without just giving answers.

50–55

XP log

Log XP, one thing that worked, one thing that was boring, and one thing to improve tomorrow.

Starter prompt

Act as a teen AI learning coach. Help me design a fun AI learning mission for a 9th grader who likes LEGO, basketball, golf, e-bikes, videos, Spanish, and algebra. Make it practical, not lecture-based. Give me 10 mission ideas, each with a build output and XP score.

Done means done

  • Sahil has a founder profile.
  • One prompt card exists.
  • One screenshot or note is saved.
  • XP is logged.
  • Tomorrow’s mission is chosen.
