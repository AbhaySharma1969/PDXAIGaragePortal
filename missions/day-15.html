Day 15 · 70 XP

Mission: School Tool Demo Day

Choose the most useful school helper from Week 3, test it, improve it, and present it as a mini product.

Time

55 min

Short, focused, and useful.

AI Skill

Product testing

One school-safe skill.

Output

Best school tool demo

A practical study artifact.

Rule

Teach, don't cheat

AI supports learning, not dishonesty.

Mission timeline

0–10

Pick best tool

Choose Spanish vocab coach, Spanish conversation bot, algebra tutor, or study guide generator.

10–25

Test with real material

Use a real class topic or practice problem.

25–40

Improve version 2

Fix anything that was too easy, confusing, or too answer-giving.

40–50

Build demo story

Explain the problem, tool, result, and improvement.

50–55

XP log

Record what school tool should be added to the final showcase.

Starter prompt

Create a 2-minute demo script for my school AI tool. Include: what problem it solves, how it helps me learn without cheating, one example of it working, one weakness, and what I improved in version 2.

Done means done

  • Best school tool demo exists.
  • One test example is completed.
  • One screenshot or note is saved.
  • One thing to improve is recorded.
  • 70 XP is logged.
