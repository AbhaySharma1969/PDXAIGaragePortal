Day 9 · 65 XP

Mission: Write Both Sides Fairly

Practice writing a fair summary of both sides without pretending to know more than the evidence supports.

Time

55 min

Keep it focused and practical.

AI Skill

Balanced writing

One skill, used immediately.

Output

Two-sided memo

Something visible by the end.

Proof

Screenshot + log

Evidence of progress.

Mission timeline

0–10

Define both sides

Identify the main positions or stakeholder perspectives.

10–25

Strongest fair version A

Write the strongest fair version of the first side.

25–40

Strongest fair version B

Write the strongest fair version of the other side.

40–50

Add uncertainty section

List what evidence would change the analysis.

50–55

XP log

Record whether the memo feels fair to both sides.

Starter prompt

Using only the evidence below, write a fair two-sided memo. Section 1: strongest fair summary of side A. Section 2: strongest fair summary of side B. Section 3: facts still missing. Section 4: what would change the conclusion.

Done means done

  • Two-sided memo exists.
  • One screenshot or note is saved.
  • One thing that worked is recorded.
  • One thing to improve is recorded.
  • 65 XP is logged.
