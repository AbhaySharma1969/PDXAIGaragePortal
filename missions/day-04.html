Mission: Personal AI Assistant Setup

Design an AI helper that supports Sahil’s learning and interests without replacing his own thinking.

Time

55 min

Keep it focused and practical.

AI Skill

Custom instructions

One skill, used immediately.

Output

Teen helper prompt

Something visible by the end.

Proof

Screenshot + log

Evidence of progress.

Mission timeline

0–10

Define the assistant

List what the assistant should help with: Spanish, algebra, LEGO, sports, e-bikes, videos.

10–25

Set rules

Create rules like: teach first, ask questions, no cheating, explain simply.

25–40

Test it

Try the assistant on one Spanish and one algebra example.

40–50

Improve it

Revise based on what felt too easy, too boring, or too confusing.

50–55

XP log

Save the final assistant instruction.

Starter prompt

Act as my personal AI learning coach. I am a 9th grader who likes LEGO, basketball, golf, e-bikes, videos, Spanish, and algebra. Teach me step by step, ask questions before giving final answers, and help me build useful things.

Done means done

  • Teen helper prompt exists.
  • One screenshot or note is saved.
  • One thing that worked is recorded.
  • One thing to improve is recorded.
  • 45 XP is logged.
