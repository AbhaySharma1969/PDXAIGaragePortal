Day 12 · 55 XP

Mission: Spanish Conversation Bot

Use AI to practice short Spanish conversations in a realistic scenario while getting corrections after each response.

Time

55 min

Short, focused, and useful.

AI Skill

Roleplay practice

One school-safe skill.

Output

Spanish roleplay script

A practical study artifact.

Rule

Teach, don't cheat

AI supports learning, not dishonesty.

Mission timeline

0–10

Choose a scenario

Pick school, restaurant, sports practice, travel, or talking with a friend.

10–25

Create roleplay rules

Tell AI to stay in character, use beginner Spanish, and correct after each answer.

25–40

Run conversation

Complete at least 8 back-and-forth lines in Spanish.

40–50

Rewrite improved version

Use the corrections to create a clean final dialogue.

50–55

XP log

Record the sentence Sahil improved the most.

Starter prompt

Act as a beginner Spanish conversation partner. Scenario: [scenario]. Use Spanish 1–2 level language. Ask me one question at a time in Spanish. After I answer, correct my grammar gently in English, then continue the conversation. At the end, create a polished version of the dialogue.

Done means done

  • Spanish roleplay script exists.
  • One test example is completed.
  • One screenshot or note is saved.
  • One thing to improve is recorded.
  • 55 XP is logged.
