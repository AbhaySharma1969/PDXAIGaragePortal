Mission: Facts vs Claims vs Opinions

Start the LEGO Case File by learning how to separate verified facts, allegations, opinions, and emotional language.

Time

55 min

Keep it focused and practical.

AI Skill

Evidence sorting

One skill, used immediately.

Output

Evidence board

Something visible by the end.

Proof

Screenshot + log

Evidence of progress.

Mission timeline

0–10

Set the standard

Agree that the mission is not gossip. The job is to organize information fairly.

10–25

Sort sample statements

Create four buckets: verified fact, claim/allegation, opinion, and emotional language.

25–40

Use AI as a sorter

Ask AI to sort the same statements, then challenge anything it gets wrong.

40–50

Create the first case board using clear labels and uncertainty notes.
