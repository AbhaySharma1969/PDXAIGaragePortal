Day 6 · 55 XP

Mission: Facts vs Claims vs Opinions

Start the LEGO Case File by learning how to separate verified facts, allegations, opinions, and emotional language.

Time

55 min

Keep it focused and practical.

AI Skill

Evidence sorting

One skill, used immediately.

Output

Evidence board

Something visible by the end.

Proof

Screenshot + log

Evidence of progress.

Mission timeline

0–10

Set the standard

Agree that the mission is not gossip. The job is to organize information fairly.

10–25

Sort sample statements

Create four buckets: verified fact, claim/allegation, opinion, and emotional language.

25–40

Use AI as a sorter

Ask AI to sort the same statements, then challenge anything it gets wrong.

40–50

Build evidence board v1

Create the first case board using clear labels and uncertainty notes.

50–55

XP log

Write one sentence about why labels matter before making conclusions.

Starter prompt

Review these case notes: [paste notes]. Create a table with four columns: verified facts, claims/allegations, opinions, and emotional language. Add a fifth column called “what proof would confirm this?” Do not make final conclusions.

Done means done

  • Evidence board exists.
  • One screenshot or note is saved.
  • One thing that worked is recorded.
  • One thing to improve is recorded.
  • 55 XP is logged.
