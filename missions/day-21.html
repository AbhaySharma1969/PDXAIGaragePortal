Mission: LEGO Price Estimator

Prototype a LEGO resale value score based on rarity, condition, completeness, demand, and confidence level.

Time

55 min

One focused build sprint.

AI Skill

Structured scoring

Practice it immediately.

Output

LEGO value scorecard

Something useful by the end.

Proof

Show what changed or improved.

Starter prompt

Create a LEGO resale value scorecard. Factors: rarity, condition, completeness, box/manuals, demand, and recent comparable prices. Use 1–5 scores and include a confidence rating. Do not invent prices; mark missing data clearly.

Done means done

  • LEGO value scorecard exists.

  • One screenshot or note is saved.
  • One weakness or improvement is recorded.
  • 60 XP is logged.
