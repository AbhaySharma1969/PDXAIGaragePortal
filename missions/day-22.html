Week 5 · Day 22 · 55 XP

Mission: E-bike Comparison Tool

Build a weighted decision matrix for comparing e-bikes based on range, speed, safety, budget, and use case.

Time

55 min

One focused build sprint.

AI Skill

Decision matrix

Practice it immediately.

Output

E-bike comparison table

Something useful by the end.

Proof

Demo note

Show what changed or improved.

Mission timeline

0–10

Define criteria

Choose what matters most: range, speed, brakes, battery, price, warranty, comfort.

10–25

Set weights

Give each criterion a weight based on importance.

25–40

Compare options

Score 3 sample e-bikes.

40–50

Explain recommendation

Write why the top option wins and what data is missing.

50–55

XP log

Save the comparison template.

Starter prompt

Create a weighted e-bike comparison matrix. Criteria: [criteria]. Options: [options]. Score each 1–5, apply weights, explain the winner, and list missing information before making a final recommendation.

Done means done

  • E-bike comparison table exists.
  • One realistic test is completed.
  • One screenshot or note is saved.
  • One weakness or improvement is recorded.
  • 55 XP is logged.
