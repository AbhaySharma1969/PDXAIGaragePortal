Mission: Golf Swing Reflection Tool

Turn golf practice into a feedback loop using notes, patterns, and next-step questions.

Time

55 min

One focused build sprint.

Practice it immediately.

Starter prompt

Create a golf practice reflection log for a teen player. Include focus area, what I tried, what happened, pattern noticed, one adjustment, and one question for next practice. Keep it simple enough to use after every session.
