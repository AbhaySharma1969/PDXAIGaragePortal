# PDX AI Garage PRD v1.2

## Product
PDX AI Garage — Founding Cohort Edition

## Audience
Teen builders ages 14–18 and their parents.

## Core promise
Teenagers learn to think, build, question, and improve with AI by completing missions that produce portfolio artifacts.

## Positioning
Not another AI course. A project-based AI apprenticeship.

## First success metric
A teenager voluntarily wants to come back for the next mission.

## Core experience
1. Welcome, Builder.
2. Choose Builder Identity.
3. Build Impossible List.
4. Complete Journey Zero.
5. Enter AI Detective Bureau.
6. Build first case file.
7. Save reflection.
8. Earn passport stamp.
9. View dashboard progress.

## Differentiators
- Builder-first language
- Garage Universe and districts
- AI Detective Bureau
- Garage Passport
- Founder Notebook
- Human judgment in every mission
- Portfolio over quizzes
- Version 2 culture

## v1.2 scope
Static public launch package with local browser progress saving. No authentication, payments, or backend yet.

## Out of scope
- Real login
- Database
- Payment
- AI API integration
- Email automation
- Live cohort management

## Next recommended version
v1.3 should add Journey Two: Builder District, connect the join form, and add a parent guide.
