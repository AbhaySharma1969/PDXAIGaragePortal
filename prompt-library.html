Reusable AI Prompts

Prompt library for the Garage.

These are starter prompts. Sahil should edit them, test them, and improve them as part of the build.

Universal

The Useful Answer Prompt

Act as a helpful coach. My goal is [goal]. My current level is [beginner/intermediate]. Give me a clear answer, but first ask up to 3 questions if you need context. Use simple language, give examples, and end with one action I can take today.
School

Teach Me, Don’t Cheat Prompt

Help me learn this without just giving me the answer. Ask me one question at a time. Give hints before solutions. When I make a mistake, explain the mistake simply and let me try again.
Investigation

Facts vs Claims Prompt

Review the notes below. Create a table with three columns: verified facts, claims/allegations, and opinions/emotional language. Mark anything uncertain. Do not make conclusions beyond the evidence.
Sports

Training Plan Prompt

Create a realistic 7-day training plan for [basketball/golf]. I have [time] per day. My goal is [goal]. Include warmup, drill, reflection question, and one measurable target.
Product

Prototype Critic Prompt

Act as a product tester for a teen AI learning portal. Review this idea/page. Tell me what is clear, what is boring, what is confusing, and what one change would make it more useful.
Showcase

Demo Story Prompt

Help me create a 3-minute demo script with this structure: problem, what I built, how AI helped, what went wrong, what I learned, and what I would improve next.
