Lab 6

Build With AI: idea to prototype.

The real skill is not just asking questions. It is using AI to build something, test it, improve it, and explain it.

1. Define

Who is the user? What problem are we solving?

2. Draft

Ask AI to create a first version, not the final version.

3. Test

Try one realistic example and look for weakness.

4. Improve

Use feedback to make version 2.
