Coach prompt

Create a LEGO resale value scorecard. Factors: rarity, condition, completeness, box/manuals, demand, and recent comparable prices. Use 1–5 scores and include a confidence rating. Do not invent prices; mark missing data clearly.

Session output

By the end of this session, create:

  • Artifact: LEGO value scorecard and test example.
  • Proof: one screenshot, note, prompt, table, or mini-demo
  • Reflection: what worked, what failed, and what version 2 should improve
Prototype rule: A prototype can estimate, but it must label missing data and confidence.
