Teaching Guide

Teach like a bootcamp, not a worksheet.

Use short sessions, visible artifacts, feedback, and a portfolio. The goal is active building, not passive watching.

Session structure

5 min — Mission goal
Name the artifact.
8 min — Learn + check
Explain idea and pass quick check.
12 min — AI practice
Use prompt and external lab if relevant.
20 min — Build
Create the artifact and test it.
5 min — Portfolio save
Save artifact, reflection, version 2 note.

Mentor questions

What did you build?How did you test it?What did AI get wrong or make vague?What did you decide yourself?What goes in your portfolio?
