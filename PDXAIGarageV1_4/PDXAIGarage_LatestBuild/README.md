# PDX AI Garage — LatestBuild

This build is designed to be uploaded into:

```text
public_html/LatestBuild/
```

Then accessed at:

```text
https://pdxaigarage.com/LatestBuild/
```

## Included

- Public landing pages
- Firebase Auth account creation
- Login/logout
- Protected Builder Dashboard
- Protected Journey Zero
- Protected Journey One
- Firestore-saved XP, identity, missions, passport stamps, and notes
- Password reset email
- Founder/Team pages
- Docs and Firebase setup guide

## Important

Before real login works, edit:

```text
assets/js/firebase-config.js
```

and paste your Firebase web app config.

## Upload rule

Upload the CONTENTS of this folder into:

```text
public_html/LatestBuild/
```

Do not upload this whole parent folder as another nested folder.
